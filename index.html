Tantalus

Final Fantasy IX Trainer

This is a simple trainer for Final Fantasy IX that automatically tracks bosses (or story-related monsters) during battle. It displays:

  • HP / MP values
  • ATB (Active Time Battle) gauges
  • Danger value
  • Encounter check indicators

How to Run

  1. Download the latest release : Tantalus Trainer - Baku Release
  2. Make sure you have a working version of Python installed
  3. Run install.bat (or manually install with pip install pymem)
  4. Start Final Fantasy IX via the Steam Launcher
  5. Run tantalus_trainer.vbs — the trainer window will appear and automatically attach to FF9
  6. Enjoy!
